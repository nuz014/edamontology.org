Eventual implementations, annotations, and collaborations with EDAM

BioCatalogue / EMBRACE Registry

The BioCatalogue is a registry of biological Web Services (and now incorporates the EMBRACE Registry of services):

  • http://www.biocatalogue.org/

There are on-going efforts to annotate the BioCatalogue web services with EDAM terms and improve ontology support within BioCatalogue.

Software Ontology

The Software Ontology (SWO) has the scope of describing types of software used in Bioinformatics. SWO covers areas such as the software type, the manufacturer of the software, the input and output data types and the uses (i.e. objectives) the software can be put to:

  • http://bioportal.bioontology.org/ontologies/42367

EDAM and SWO are complementary developments and the developers are collaborating to ensure better integration and development of both resources.

BioSharing

The BioSharing project aims to support the development of data policies and standards in the biological domain. They are investigating using EDAM for annotation of the resources included under BioSharing:

  • http://www.biosharing.org/

BioNemus

BioNemus is a graphical tool for datatype definition and WSDL development.

  • http://inb.bsc.es/applications/java/bionemus2/

Dmitry Repchevsky is investigating importing EDAM into BioNemus.

Ondex

The Ondex data integration platform enables data from diverse biological data sets to be linked, integrated and visualised through graph analysis techniques:

  • http://www.ondex.org/

Ondex are investigating using EDAM for annotation.

WHAT-IF service annotations

WHAT-IF is a molecular modelling package for proteins and associated molecules. It's functionality is available as web-services:

  • http://swift.cmbi.ru.nl/whatif/

The WHAT-IF developers are using EDAM in the annotation of their services.

IBCP cloud web services

Some web services hosted at IBCP, CNRS, Lyon that run on a cloud are annotated with EDAM terms, either directly or via their use of BioXSD:

  • http://gbio-pbil.ibcp.fr/Tools/

CBS DTU resource annotation

Kristoffer Rapacki is investigating using EDAM for the annotation of resources hosted by the Center for Biological Sequence Analysis, Technical University of Denmark. Some web services hosted at CBS are annotated by EDAM terms, either directly or via their use of BioXSD.

  • http://www.cbs.dtu.dk/
